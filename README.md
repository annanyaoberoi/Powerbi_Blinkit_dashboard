# Blinkit Sales Data Analysis Dashboard

## Project Overview
This project focuses on analyzing Blinkit sales data using Power BI to gain insights into sales performance, product trends, and customer behavior. The dashboard provides interactive visualizations to support data-driven decision-making.

## Objectives
- Analyze overall sales and revenue trends
- Identify top-performing products and categories
- Monitor customer purchasing patterns
- Provide insights for business improvement

## Tools and Technologies Used
- Power BI
- Excel
- Data Visualization
- Data Analysis

## Key Features
- Interactive dashboard with filters and slicers
- Visual representation of sales trends and KPIs
- Category-wise and product-wise performance analysis
- Easy-to-understand charts and graphs

## Files Included
- Blinkit_Dataset.xlsx (Sales dataset)
- Blinkit_Dashboard.pbix (Power BI dashboard file)
- Dashboard_Screenshot.png (Dashboard preview)

## Conclusion
This project demonstrates the use of Power BI to transform raw data into meaningful insights and support business decision-making through effective data visualization.
